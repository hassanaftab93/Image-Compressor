  ```bash
  cd /Path/To/Project
  ```
  
  Create a Virtual Environment for this Project

  ```bash
  python -m venv venv
  ```

  Activate Virtual Environment for this Project

  Windows:
  ```bash
  source venv/Scripts/activate
  ```

  ```bash
  source ./venv/bin/activate
  ```

